# PlaneWar 飞机大战游戏

## 简介

一款经典的飞机大战游戏，基于`Java Swing`实现。

总共含有四个关卡，每关均有boss，难度依次递增。击毁敌机有一定机率出现奖励，奖励主要有三种：加血，增加大招次数和升级飞机炮弹。

## 操作

鼠标控制飞机移动，鼠标左键放大招。
